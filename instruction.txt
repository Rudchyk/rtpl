Создать расширение
    gem build rtpl-compass.gemspec

Отправить расширение в Ruby gem:
    gem push rtpl-compass-VERSION.gem

Установить расширение локально:
    gem install rtpl-compass-VERSION.gem

Установить проект с Ruby gem:
    gem install rtpl-compass

Удалить расширение с Ruby gem:
    gem yank rtpl-compass -v VERSION
